# RFP Management Application

React application for managing RFP (Request for Proposal) rooming lists with advanced search, filtering, and collapsible event grouping features.

## Features

- **Real-time Search** - Search across RFP names and agreement types with debouncing
- **Advanced Filtering** - Filter events by status with multi-select support
- **Event Grouping** - Organized by event name with color-coded sections
- **Collapsible Views** - Three display modes: Expanded, One Row (horizontal scroll), and Collapsed
- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Smooth Animations** - Polished transitions and staggered card animations
- **Material Design** - Clean, modern UI with custom color palette
